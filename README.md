# Kalkulator modelarski v1.2

Aplikacja webowa do używania na Windowsie w Brave/Chrome.

## Uruchamianie lokalne

1. Rozpakuj ZIP.
2. Otwórz plik `index.html` dwuklikiem.

## GitHub Pages

1. Wrzuć wszystkie pliki z tej paczki do repozytorium.
2. Settings → Pages.
3. Deploy from a branch → main → /root.

## Poprawki w tej wersji

- działające przyciski,
- panel pod wynikiem działa jako status,
- brak dodatkowych wartości/wzorów pod wynikiem,
- komunikaty statusu:
  - Narzędzie gotowe do obliczeń.
  - Dane zapisane w historii.
  - Wynik skopiowany do schowka.
  - Wszystkie pola zostały wyczyszczone.
  - Uzupełnij wszystkie wymagane pola.
  - Wprowadzono nieprawidłowe dane.
- subtelny niebieski błysk wyniku po obliczeniu,
- domyślny skurcz 1%,
- wynik zawsze z 8 miejscami po przecinku.
